# Lova Themes

個人 VS Code 色彩主題擴展。

## 主題

### Lova Dark Green

深色主題，以青綠色 (#4EC9B0) 為強調色，搭配 Monokai 風格的語法高亮。

## 本地安裝

```bash
./ln.sh
```

重新啟動 VS Code 後，`Cmd+K Cmd+T` 選擇 **Lova Dark Green**

## 色彩配置

| 元素 | 色彩 |
|------|------|
| 強調色 | #4EC9B0 |
| 背景 | #2f2f2f |
| 側邊欄 | #353535 |
| 關鍵字 | #F92672 |
| 函數 | #A6E22E |
| 字串 | #E6DB74 |
| 型別 | #66D9EF |
| 數字 | #AE81FF |
| 參數 | #FD971F |
| 註解 | #75715E |
