# Lova Themes

個人 VS Code 色彩主題擴展，包含 19 個主題（9 深色 + 10 淺色）。

## 主題列表

### 深色主題

| 主題 | 說明 |
|------|------|
| Lova Monokai Green | 青綠色強調的 Monokai 風格 |
| Lova Ocean Depths | 深海藍調 |
| Lova Tech Innovation | 科技感配色 |
| Lova Midnight Galaxy | 午夜星空 |
| Lova Forest Canopy | 森林綠意 |
| Lova Sunset Boulevard | 日落暖調 |
| Lova Claude Dark | Claude AI 深色版 |
| Lova Teams Dark | Microsoft Teams 深色版 |
| Lova shadcn Dark | shadcn/ui 深色版 |

### 淺色主題

| 主題 | 說明 |
|------|------|
| Lova Cursor Light | Cursor 編輯器風格 |
| Lova Arctic Frost | 冰霜白調 |
| Lova Claude | Claude AI 淺色版 |
| Lova Teams | Microsoft Teams 風格 |
| Lova macOS Light | macOS 原生風格 |
| Lova iOS Light | iOS 原生風格 |
| Lova Android Light | Android Material 風格 |
| Lova Nintendo Light | Nintendo 紅白配色 |

| Lova shadcn Light | shadcn/ui 淺色版 |

## 安裝

### 本地安裝

```bash
./ln.sh
```

重新啟動 VS Code 後，`Cmd+K Cmd+T` 選擇主題。
