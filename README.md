# Lova Themes

個人 VS Code 色彩主題擴展，包含 18 個主題（9 深色 + 9 淺色）。

## 主題列表

| 主題             | Light | Dark | 說明                      |
| ---------------- | :---: | :--: | ------------------------- |
| Monokai Green    |  ✅   |  ✅  | 青綠色強調的 Monokai 風格 |
| Ocean Depths     |  ✅   |  ✅  | 深海藍調                  |
| Tech Innovation  |  ✅   |  ✅  | 科技感配色                |
| Midnight Galaxy  |  ✅   |  ✅  | 午夜星空                  |
| Arctic Frost     |  ✅   |  ✅  | 冰霜色調                  |
| Forest Canopy    |  ✅   |  ✅  | 森林綠意                  |
| Sunset Boulevard |  ✅   |  ✅  | 日落暖調                  |
| Terminal         |  ✅   |  ✅  | 終端機風格                |
| Crimson Ember    |  ✅   |  ✅  | 酒紅餘燼色調              |

## 安裝

### 本地安裝

```bash
npm run install:ext
```

重新啟動 VS Code 後，`Cmd+K Cmd+T` 選擇主題。
